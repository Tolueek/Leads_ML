SuperPass Leads Classification — Submission Package
Date: 2025-09-18

Included files:
1) superpass_leads_300.csv — Synthetic dataset (300 rows)
2) SuperPass_Leads_Classification.ipynb — End-to-end notebook (preprocessing, modeling, evaluation)

How to run:
- Open the notebook in Jupyter/Colab and run all cells.
- The notebook reads the CSV, trains models, and shows metrics & plots.
